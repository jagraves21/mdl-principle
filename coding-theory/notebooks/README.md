# Coding Theory Notebooks

This directory contains interactive Jupyter notebooks that demonstrate the coding theory concepts introduced in the [../theory](../theory/) directory.

## Code Examples

These notebooks illustrate the main integer and prefix-free codes that form the foundation of many advanced codes:

- [**01-unary-code.ipynb**](01-unary-code.ipynb) &mdash; Demonstrates unary coding and its simplicity as a baseline integer code.
- [**02-elias-gamma-code.ipynb**](02-elias-gamma-code.ipynb) &mdash; Demonstrates Elias gamma coding, showing how integers can be efficiently represented.
- [**03-elias-delta-code.ipynb**](03-elias-delta-code.ipynb) &mdash; Demonstrates Elias delta coding, illustrating its connection to Elias gamma coding.
- [**04-elias-omega-code.ipynb**](04-elias-omega-code.ipynb) &mdash; Demonstrates Elias omega coding, illustrating recursive integer encoding.
- [**05-fibonacci-code.ipynb**](05-fibonacci-code.ipynb) &mdash; Demonstrates Fibonacci coding, an alternative universal integer code to Elias codes.

## Applied Algorithms

These notebooks implement fundamental algorithms in coding theory, showing how codes can be constructed and analyzed in practice:

- [**06-huffman-code.ipynb**](06-huffman-code.ipynb) &mdash; Builds Huffman codes from symbol probabilities, demonstrating optimal prefix-free coding.
